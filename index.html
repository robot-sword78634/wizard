About the Game

Wizard is a C++ console game where you battle an evil wizard using attacks and defense strategies. Keep your hit points above 0 while defeating the evil wizard to win.

How to Play Locally

  1. Install a C++ compiler (like GCC or Visual Studio).
  2. Copy the wizard.cpp file to your computer.
  3. Compile it: g++ wizard.cpp -o wizard
  4. Run the game: ./wizard (Linux/Mac) or wizard.exe (Windows)

Game Code


#include <iostream>
using namespace std;

int main() {
    srand(time(0));
    int life = 100;
    int enemy_life = 125;
    bool ans = 0;

    cout << "Welcome to Wizard!" << endl;

    while(enemy_life > 0 && life > 0) {
        cout << "Choose action: (0) Fight, (1) Defend: ";
        cin >> ans;
        if(ans == 0) {
            int chance = rand() % 100;
            if(chance < 52) life -= 12;
            enemy_life -= 10;
            cout << "YOUR HP: " << life << " | ENEMY HP: " << enemy_life << endl;
        } else {
            cout << "You defend successfully!" << endl;
        }
    }

    cout << "Final HP: " << life << endl;
    if(enemy_life < 1) cout << "You defeated the Evil Wizard!" << endl;
    else cout << "The Evil Wizard prevails..." << endl;
}
